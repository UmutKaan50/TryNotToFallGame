# TryNotToFallGame

Controls:

⬆-⬇/ w-s : move up/down

⬅-➡/a-d : move and rotate camera

Powerups make your ball much stronger. ⚡

Aim:

Try not to fall (as the name suggests) for as high levels as possible.
